# Build interface just like Zoo Tycoon with extracted assets

The goal is to recreate the user interface of the original Zoo Tycoon (2001) using the assets extracted from the game files. This includes:
- Main menu and sub-menus.
- Toolbars for construction (fences, animals, scenery, etc.).
- Entity information panels.
- Game status indicators (funds, guest happiness, animal happiness).
- Custom cursors and buttons using original textures.
