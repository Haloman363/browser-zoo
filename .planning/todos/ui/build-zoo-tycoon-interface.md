# Build interface just like Zoo Tycoon with extracted assets

The goal is to recreate the user interface of the original Zoo Tycoon (2001) using the assets extracted from the game files.

## Progress
- [x] Main menu with authentic background and music.
- [x] HUD Toolbars (Left/Bottom bars) with original assets.
- [x] Selection Catalog (objpan) for animals, scenery, etc.
- [x] Money display integrated with HUD.
- [x] Entity information panels (Animal inspection window).
- [x] Game status indicators (happiness meters in HUD and Panels).
- [x] Custom cursors (hand, grabber, paint, bulldozer).
- [x] Sound effects for all UI interactions.
- [x] Scenario selection flow.
