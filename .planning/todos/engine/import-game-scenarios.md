# Import existing game scenarios

The goal is to implement a system to import and parse the original Zoo Tycoon scenario files. This includes:
- Decoding scenario metadata (goals, constraints, starting funds).
- Loading specific map layouts and entity placements.
- Implementing the scenario logic (win/loss conditions).
- Integrating scenarios into the main menu/game selection UI.
